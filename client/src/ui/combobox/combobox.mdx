## Usage

```tsx
<Combobox
	options={["Next.js", "Astro", "Qwik", "SolidStart", "Nuxt.js"]}
	placeholder="Search framework..."
	itemComponent={props => (
		<ComboboxItem item={props.item}>{props.item.rawValue}</ComboboxItem>
	)}>
	<ComboboxTrigger>
		<ComboboxInput />
	</ComboboxTrigger>
	<ComboboxContent />
</Combobox>
```
